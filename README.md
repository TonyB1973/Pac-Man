# Maze Munch PWA

A complete offline retro maze-chase game.

## Run locally
Serve the folder with any local web server. Service workers do not run reliably from file:// URLs.

## GitHub Pages
Upload the folder contents to a GitHub repository, enable Pages from the main branch/root, then open the Pages URL.

## Controls
- Arrow keys or WASD
- Swipe on the game area
- On-screen directional pad

## Included
- Installable PWA
- Offline service worker
- Touch controls
- Score, high score, levels and lives
- Power orbs and enemy chase behaviour
- Responsive phone/tablet/desktop layout
