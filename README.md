# Maze Munch 2

A complete original retro maze-chase PWA.

## Version 2 changes
- New PWA identity and cache name to avoid the Version 1 cache
- True drag-and-hold virtual joystick
- Buffered cornering at maze intersections
- Scripted timed exits from the centre house
- Smooth 60 FPS canvas animation
- Responsive full-screen mobile layout
- Keyboard, swipe and joystick controls
- Offline installation support

## Publishing
Upload all files and folders to the root of a GitHub repository and enable GitHub Pages.

Because this is a new PWA identity, remove the old Home Screen copy before installing Maze Munch 2.
